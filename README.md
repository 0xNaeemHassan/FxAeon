# FxAeon

Telegram DeFi trading bot for f(x) Protocol on Ethereum mainnet.

## Repository

- **GitHub**: https://github.com/0xNaeemHassan/FxAeon
- **Bot**: [@FxAeonBot](https://t.me/FxAeonBot)

## Quick Start

```bash
# Clone the repo
git clone https://github.com/0xNaeemHassan/FxAeon.git
cd FxAeon

# Setup (creates .env files from templates)
./scripts/dev-setup.sh

# Edit .env files with your credentials
# Then deploy
docker-compose up -d
```
